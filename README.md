<hr>

<h2>Description</h2>
<p>
  <b>EC (Elemental Clock)</b> displays real-time Vana'diel information in a clean, compact UI line.  
  It shows the current in-game <b>time</b>, <b>weather</b>, <b>day</b>, <b>moon phase</b>,  
  <b>moon trend</b>, and the day's <b>elemental weakness</b>.  
  All elements support tooltips and offer full customization for fonts, scale, visibility, and opacity.
</p>

<hr>

<h2>Features</h2>
<ul>
  <li>Current Vana'diel time</li>
  <li>Live weather with icon + tooltip</li>
  <li>Day icon + optional colored text</li>
  <li>Moon phase icon + percent + tooltip</li>
  <li>Moon trend (waxing/waning arrow)</li>
  <li>Weakness element (icon + text)</li>
</ul>

<h3>Adjustable Options</h3>
<ul>
  <li>Window opacity</li>
  <li>Content scale</li>
  <li>Font and font size</li>
  <li>Show/hide individual elements</li>
  <li>Persistent settings saved to <code>addons/ec/settings.lua</code></li>
  <li>Uses ImGui for clean UI rendering</li>
</ul>

<hr>

<h2>Commands</h2>
<pre>/ec settings</pre>
<p>Opens or closes the EC settings window.</p>
